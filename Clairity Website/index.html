More Features

Learn about how you can fully utilize your Clairity air filtration system.

Energy Efficient Design

The Clairity air filtration system only runs when it needs to, which means it wont be eating up power!

Filtration Schedules

Using modern AI technology, the Clairity filtration system will create a schedule in which it can decide when it should and shouldn't start filtering out the air.

Emergency Auto-Dial

If the Clairity air filtration system detects a dangerous amount of hazardous particles, it will send a notification to the app and dial emergency services if anyone is detected in the house and unresponsive.

Offline Functionality

The Clairity air filtration system will still function independently even if your wifi goes out, and it will send all data to the app once it is back online.
